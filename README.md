# Project : 숫자 맞추기 게임(number guessing game)

### 구현 목표
입력칸에 숫자를 입력하고 확인을 눌렀을때 정답을 맞추면 게임이 끝나는 게임으로 맞추지 못하면 입력한 숫자들을 예측한 숫자들 옆에 계속 보여주고 남은 기회가 몇번 남았는지를 알려주게 된다. 또한, 정답에 얼마나 가깝고 먼지 직관적으로 보여주기 위해 원을 통해 추론을 도와준다.
